## [Infix to Postfix Expression](#)

Easy

Given an infix expression, convert it to the postfix expression. Assume that the infix expression is a string of tokens without any whitespace.

Example 1:

Input: "A*B+C"

Output: "AB*C+"

Example 2:

Input: "(A+B)*(C/D)"

Output: "AB+CD/*"

Example 3:

Input: "(A+B)*C+(D-E)/F+G"

Output: "AB+C*DE-F/+G+"
